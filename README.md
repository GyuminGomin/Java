# JDK 설치
1. OpenJDK 검색
1. OpenJDK 11 버전 검색
1. 또는zuju jdk 11 검색
1. 사이트에 들어가, Java11(LTS)설치 (윈도우) .zip으로해도 되고, .msi로 해도 됨
1. 환경 변수 설정 한 후, cmd로 확인 하면 끝

# Maven 설치
1. maven 다운로드 검색
1. 다운로드 후 압축풀기 해서 환경변수 설정 bin폴더로
1. vscode Extension Pack for Java를 설치하면 끝
1. Extension에서 Community Server Connection 설치
